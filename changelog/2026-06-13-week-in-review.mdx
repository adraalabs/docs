A big week for Adraa Inbox: the **public REST API** is now live, and you can drive most of your workspace from your own systems.

## New

- **Public REST API.** A versioned REST API at `https://api.inbox.adraa.ai`, authenticated with workspace API tokens. Start with the [API introduction](/api-reference/introduction) and the [quickstart](/quickstart).
- **Workspace API tokens.** Admins can create and manage tokens in **Settings → API**. See the [authentication guide](/authentication).
- **Conversations API.** List conversations with cursor pagination and console-style filters, assign teammates or the workspace AI agent, and send messages from your own systems.
- **Contacts API.** List and filter contacts with the same options as the console's Contacts page, fetch full contact profiles (including flow history and internal notes), and merge custom fields to enrich contacts from your CRM.
- **Agents API.** List, invite, update, and remove workspace agents programmatically — useful for syncing teammates from your HR or onboarding tools. Removing the last admin is blocked so you can't lock yourself out.

## Updates

- **Guide in the navbar.** The product **Guide** is now linked directly from the top navigation, so it's one click from any page.

## Get started

Generate a workspace API token from **Settings → API**, follow the [authentication guide](/authentication), and make your first call from the [quickstart](/quickstart).
